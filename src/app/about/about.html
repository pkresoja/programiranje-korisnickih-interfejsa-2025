Programiranje Korisničkih Interfejsa

Aplikacija napravljena tokom vežbi u oktobarskom bloku iz istoimenog predmeta.

Aplikacija koristi sledeće tehnologije:

  • Angular 20
  • Bootstrap 5.3 - Stil
  • Sweet Alerts 2 - Dijalozi
  • Fonts Awesome 7 - Ikonice
  • Axios HTTP Client
  • TypeScript
  • Flights API
